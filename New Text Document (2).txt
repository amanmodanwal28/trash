
### **1. Set Theory and Relations**
1. Define **set, subset, power set, universal set, and complement of a set** with examples.
2. Prove **De Morgan’s laws** for sets.
3. What is a **Cartesian Product**? Find **A × B** where \( A = \{1,2\}, B = \{a,b\} \).
4. Define **equivalence relation** and prove that a given relation is an **equivalence relation**.
5. Find the **reflexive, symmetric, and transitive closure** of a relation.

---

### **2. Functions**
1. Define **one-to-one (injective), onto (surjective), and bijective** functions with examples.
2. Prove that the **composition of two bijective functions** is also bijective.
3. What are **inverse functions**? Prove that if a function is **bijective, then its inverse exists**.
4. Define **recursively defined functions** with an example.

---

### **3. Propositional Logic and Predicate Logic**
1. Write **truth tables** for **AND, OR, XOR, NAND, NOR**.
2. Prove **p → q ≡ ~p ∨ q** using a truth table.
3. Convert a given **propositional logic statement** into **CNF (Conjunctive Normal Form) or DNF (Disjunctive Normal Form)**.
4. Prove **tautology and contradiction** with examples.
5. What is **predicate logic**? Explain **universal quantifier (∀) and existential quantifier (∃)** with examples.
6. Convert the given **statement into predicate logic**:  
   *"All students in the class are intelligent."*

---

### **4. Mathematical Induction & Recurrence Relations**
1. Prove by **mathematical induction**:  
   \( 1 + 3 + 5 + \dots + (2n - 1) = n^2 \).
2. Solve the **recurrence relation**:  
   \( T(n) = 2T(n/2) + n \) using **Master’s Theorem**.
3. Find the **solution of Fibonacci recurrence relation**:  
   \( F(n) = F(n-1) + F(n-2), F(0) = 0, F(1) = 1 \).

---

### **5. Counting and Combinatorics**
1. What is the **Pigeonhole Principle**? Prove an example.
2. Find the number of ways to **select 3 students from a group of 10**.
3. Prove:  
   \[
   C(n, r) = C(n-1, r) + C(n-1, r-1)
   \]
4. How many ways can 5 **different books** be arranged on a shelf?
5. Solve a **binomial theorem problem**:  
   Expand \( (x + y)^5 \) using binomial expansion.

---

### **6. Graph Theory**
1. Define **simple graph, multigraph, and complete graph**.
2. Prove that the **sum of degrees of all vertices is twice the number of edges**.
3. Define **Eulerian path and Hamiltonian path** with examples.
4. Find the **chromatic number** of a given graph.
5. Explain **DFS and BFS** algorithms with an example.

---

### **7. Trees**
1. Define **tree, binary tree, and binary search tree (BST)**.
2. Prove that **a tree with n vertices has (n-1) edges**.
3. Find the **height of a binary tree**.
4. Explain the **preorder, inorder, and postorder** traversal of a binary tree.
5. What is a **spanning tree**? Explain **Prim’s and Kruskal’s Algorithm**.

---

### **8. Boolean Algebra**
1. State and prove **De Morgan’s Theorems**.
2. Simplify a Boolean function using **K-map**.
3. Convert the given **Boolean function into a logic circuit**.
4. Explain the difference between **SOP (Sum of Products) and POS (Product of Sums)** form.
5. Design a **half-adder and full-adder** using logic gates.
